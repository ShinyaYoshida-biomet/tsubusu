# Tsubusu

A minimal ToDo app for macOS, inspired by Stickies but focused on task management.



## Getting Started

To run the app:

```bash
flutter run -d macos
```

## Requirements

- Flutter SDK (3.29.3 or later)
- Dart SDK (3.7.2 or later)
- macOS development environment
- Xcode (for macOS development)
